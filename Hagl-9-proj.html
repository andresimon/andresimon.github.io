Hagl-9
Project Type Student project
Technology Android / Java
Primary Role(s) Framework architect / Animations / coding

HAGL-9 is a comedic pawn store simulator that takes place in the year 25XX. The game centres around attempting to make a profit from haggling culturally relevant objects from customers at the best price so that the pawn shop can stay in business.


This was a group project, with 4 programmers, including myself, and an artist.

My contribution was:

✔ Using the basic framework provided, I created a set of classes that expanded this framework, allowing and making it easier for other programmers to create game screens by simply extending those classes.

✔ I implemented the Music & Sound Effects On/Off feature.

✔ I have also created the classes responsible for 2D animation, which use spritesheets with any number of frames.
