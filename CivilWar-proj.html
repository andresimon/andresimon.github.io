Civil War
Project Type Student project
Technology Unity
Primary Role(s) Solo Game Design / coding

Civil War is a 1-level demo of a game that honors the movie with the same name.

✔ The hero uses a 4-layer animation to reflect his movements with and without shield;

✔ There is a system of combos that makes the hero use different attacks (punches and kicks);

✔ I use the StateMachineBehavior class to create a State Machine for the characters and one for the enemies, linked to the states of the animations;

✔ Enemies have both melee and ranged attacks, and will choose which to use according to their distance from the hero;

✔ Use of object-oriented language concepts, such as interfaces and inheritance, to create a code of easy maintenance and high reusability;
