Vault 163

Vault 163 is a simple HTML / CSS game that pays tribute to the 80’s old adventures and RPG game books.

In this game, the player must solve the maze to find the exit, navigating the rooms and using text commands to perform specific actions.

Gameplay:

To move between rooms, the player must click on the arrows that are available.

The commands must be typed in the input field.

The ‘help’ command lists the verbs that can be used, as well as tips of what can be done in certain rooms.
